# 基于springboot和vue的实验室管理系统2



## 1、项目介绍

基于springboot和vue的实验室管理系统2拥有两种角色

管理员：实验室管理、用户管理、申请管理、设备管理、消耗品管理等

用户：登录注册、申请实验室、申请和报备消耗品等

## 2、项目技术

后端框架：springboot、Mybatis

前端技术：VUE、html

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：无需
- maven版本：无限制
- 硬件环境：Windows
